# JsonCrdt Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a `JsonCrdt` rung to `:kuilt-crdt` — a CRDT over arbitrary JSON that deep-merges objects and array elements, supersedes type changes, and surfaces type/scalar conflicts — by composing `ORMap`, `Rga`, and a per-node causal register.

**Architecture:** Every JSON node is a causal register `Causal<DotFun<Content>>` whose assignments are typed content (`Obj(ORMap<String, JsonNode>)`, `Arr(Rga<Unit>` order `+ ORMap<ElementId, JsonNode>` content`)`, `Scalar(JsonPrimitive)`). `piece` is the register's causal join; surviving same-kind assignments fold structurally, surviving different-kind assignments are a conflict. The register's dots give the causal dominance that makes a type-overwrite supersede rather than resurrect a spurious conflict. Full rationale: `docs/superpowers/specs/2026-06-10-json-crdt-design.md`.

**Tech Stack:** Kotlin Multiplatform, `explicitApi()` enforced (every public declaration needs an explicit modifier). kotlinx.serialization (`@Serializable`, CBOR via the replicator, `kotlinx-serialization-json` for the `JsonElement` value model). Composes existing zoo types `Causal`, `DotFun`, `DotContext`, `Dot`, `ORMap`, `Rga`, `RgaId`, `ReplicaId`. jqwik for JVM property tests.

**Key facts about the primitives this composes (verified against source):**
- `Causal<S>(store, context)`; `piece` = `store.join(other.store, ctx, otherCtx)` + `context.piece`.
- `DotFun<V>(values: Map<Dot, V>)` is a `DotStore`; its causal join keeps mutually-concurrent dots (the MVRegister core). It does **not** merge values — a dot is minted once so same dot ⇒ same value.
- `DotContext.nextDot(replica)` / `.add(dot)` / `.contains(dot)`. **One `DotContext` per CRDT instance** — nested sub-CRDTs each own their own context, all minted under the same `ReplicaId`; their dot spaces are private, so reuse of one `ReplicaId` across nested contexts is correct (this is the established nested-CRDT pattern).
- `ORMap.empty<K, S>()`, `.put(replica, key, value)`, `.remove(key)`, `.get(key)`, `.keys`, `.piece`. `S : Quilted<S>` — `JsonNode` satisfies this, so values recurse.
- `Rga.empty<V>()`, `.insertAt(replica, index, value): Pair<Rga<V>, RgaOp.Insert<V>>` (`op.id` is the element's `RgaId`), `.removeAt(index): Pair<Rga<V>, RgaOp.Remove<V>>?`, `.sequence: List<RgaId>`, `.tombstones: Set<RgaId>`, `.piece`, `Rga.wireSerializer(vSerializer)`.
- `RgaId` is `@Serializable`, `Comparable`, a `data class`. `Dot(replica, seq)` is `@Serializable`; `seq >= 1`.
- `kotlinx.serialization.json.JsonNull` **is a** `JsonPrimitive`, so `is JsonPrimitive` covers JSON null.

```kotlin
package us.tractat.kuilt.crdt

import kotlinx.serialization.json.JsonArray
import kotlinx.serialization.json.JsonElement
import kotlinx.serialization.json.JsonObject
import kotlinx.serialization.json.JsonPrimitive
import kotlin.jvm.JvmInline

/** Stable identity of an array element: the [RgaId] the order-RGA assigned at insert. */
@JvmInline
public value class ElementId(public val rgaId: RgaId)

/**
 * The typed content one assignment of a [JsonNode] can hold. Internal: callers
 * see only [JsonNode] / [JsonElement]. A closed sealed set of three kinds —
 * object, array, scalar — is what a node's register chooses between.
 */
internal sealed interface Content {
    data class Obj(val map: ORMap<String, JsonNode>) : Content
    data class Arr(val order: Rga<Unit>, val items: ORMap<ElementId, JsonNode>) : Content
    data class Scalar(val value: JsonPrimitive) : Content
}

/** Total order over surviving register entries; picks the deterministic conflict winner. */
private val DOT_ORDER: Comparator<Map.Entry<Dot, Content>> =
    compareBy({ it.key.seq }, { it.key.replica.value })

/**
 * One node of a [JsonCrdt] document: a causal register over typed [Content].
 *
 * `piece` is the register's causal join (delegated to [Causal]). Surviving
 * same-kind assignments fold structurally on materialization (objects deep-merge,
 * arrays merge order+content, scalars stay multi-valued); surviving different-kind
 * assignments are a type-conflict. The register's dots supply causal dominance, so
 * an observed overwrite (even a type change) supersedes the old assignment instead
 * of resurrecting it on re-merge — the property a bare structural sum type lacks.
 *
 * **Convention:** the empty register (`DotFun()` + `DotContext.EMPTY`) materializes
 * as the empty JSON object `{}` and is the structural zero used for the document
 * root and for auto-created intermediate objects.
 */
public class JsonNode internal constructor(
    internal val reg: Causal<DotFun<Content>>,
) : Quilted<JsonNode> {

    override fun piece(other: JsonNode): JsonNode = JsonNode(reg.piece(other.reg))

    /** Mint a fresh dot that supersedes every assignment this register has observed. */
    internal fun writeSupersede(replica: ReplicaId, content: Content): JsonNode {
        val dot = reg.context.nextDot(replica)
        return JsonNode(Causal(DotFun(mapOf(dot to content)), reg.context.add(dot)))
    }

    /** Overwrite this node with [value], superseding any prior assignment(s). */
    public fun setScalar(replica: ReplicaId, value: JsonPrimitive): JsonNode =
        writeSupersede(replica, Content.Scalar(value))

    // ---- materialization ----

    private val survivors: Map<Dot, Content> get() = reg.store.values

    private fun objSurvivors() = survivors.values.filterIsInstance<Content.Obj>()
    private fun arrSurvivors() = survivors.values.filterIsInstance<Content.Arr>()
    private fun scalarSurvivors() = survivors.entries.filter { it.value is Content.Scalar }

    /** Fold this node's surviving object assignments into one [ORMap] (empty if none). */
    internal fun objMap(): ORMap<String, JsonNode> =
        objSurvivors().map { it.map }.reduceOrNull { x, y -> x.piece(y) } ?: ORMap.empty()

    /** Fold this node's surviving array assignments into one [Content.Arr] (or null if none). */
    internal fun arrContent(): Content.Arr? =
        arrSurvivors().reduceOrNull { x, y -> Content.Arr(x.order.piece(y.order), x.items.piece(y.items)) }

    private fun materializeObj(): JsonObject {
        val map = objMap()
        return JsonObject(map.keys.associateWith { k -> map.get(k)!!.toJsonElement() })
    }

    private fun materializeArr(): JsonArray {
        val arr = arrContent() ?: return JsonArray(emptyList())
        val visible = arr.order.sequence.filter { it !in arr.order.tombstones }
        return JsonArray(visible.mapNotNull { id -> arr.items.get(ElementId(id))?.toJsonElement() })
    }

    private fun materializeScalar(): JsonPrimitive =
        (scalarSurvivors().maxWith(DOT_ORDER).value as Content.Scalar).value

    /** Materialize this node to a plain [JsonElement]; deterministic across replicas. */
    public fun toJsonElement(): JsonElement {
        if (survivors.isEmpty()) return JsonObject(emptyMap()) // empty register ≙ {}
        return when (survivors.entries.maxWith(DOT_ORDER).value) {
            is Content.Obj -> materializeObj()
            is Content.Arr -> materializeArr()
            is Content.Scalar -> materializeScalar()
        }
    }

    /**
     * The conflicting alternatives at this node: one folded value per surviving kind,
     * plus each distinct concurrent scalar. Empty when there is exactly one
     * unambiguous value (no type-conflict and no concurrent scalars).
     */
    public fun conflicts(): List<JsonElement> {
        val alts = mutableListOf<JsonElement>()
        if (objSurvivors().isNotEmpty()) alts += materializeObj()
        if (arrSurvivors().isNotEmpty()) alts += materializeArr()
        scalarSurvivors().map { (it.value as Content.Scalar).value }.distinct().forEach { alts += it }
        return if (alts.size <= 1) emptyList() else alts
    }

    override fun equals(other: Any?): Boolean = other is JsonNode && reg == other.reg
    override fun hashCode(): Int = reg.hashCode()
    override fun toString(): String = "JsonNode(${toJsonElement()})"

    public companion object {
        /** The structural zero: an empty register, materializing as `{}`. */
        public fun emptyObject(): JsonNode = JsonNode(Causal(DotFun(), DotContext.EMPTY))

        /** A scalar leaf holding [value], authored by [replica]. */
        public fun scalar(replica: ReplicaId, value: JsonPrimitive): JsonNode =
            emptyObject().setScalar(replica, value)
    }
}
```

```kotlin
    /** Put [child] under [key], asserting this node is an object (supersedes other kinds). */
    public fun put(replica: ReplicaId, key: String, child: JsonNode): JsonNode =
        writeSupersede(replica, Content.Obj(objMap().put(replica, key, child)))

    /** Remove [key], asserting this node is an object. No-op materially if absent. */
    public fun removeKey(replica: ReplicaId, key: String): JsonNode =
        writeSupersede(replica, Content.Obj(objMap().remove(key)))

    /** The child node under object [key], or null if this node is not an object with that key. */
    public fun child(key: String): JsonNode? = objMap().get(key)
```

Note: `objMap()` folds the current object survivors (or empty), so `put` over a multi-survivor or wrong-kind node first resolves to a single object base, then supersedes — the local writer's "this is an object" intent is unambiguous.

```kotlin
    /** Insert [child] at visible [index] (0 = front), asserting this node is an array. */
    public fun insertAt(replica: ReplicaId, index: Int, child: JsonNode): JsonNode {
        val base = arrContent() ?: Content.Arr(Rga.empty(), ORMap.empty())
        val (newOrder, op) = base.order.insertAt(replica, index, Unit)
        val eid = ElementId(op.id)
        return writeSupersede(replica, Content.Arr(newOrder, base.items.put(replica, eid, child)))
    }

    /** Remove the element at visible [index]; no-op if out of range or not an array. */
    public fun removeAt(replica: ReplicaId, index: Int): JsonNode {
        val base = arrContent() ?: return this
        val visible = base.order.sequence.filter { it !in base.order.tombstones }
        if (index !in visible.indices) return this
        val eid = ElementId(visible[index])
        val (newOrder, _) = base.order.removeAt(index) ?: return this
        return writeSupersede(replica, Content.Arr(newOrder, base.items.remove(eid)))
    }

    /** The [ElementId]s of this array's visible elements, in order (empty if not an array). */
    public fun arrVisibleIds(): List<ElementId> {
        val base = arrContent() ?: return emptyList()
        return base.order.sequence.filter { it !in base.order.tombstones }.map { ElementId(it) }
    }

    /**
     * Replace the element [id] by applying [edit] to it, keeping its position.
     * This is the mutate-same-identity path that makes concurrent element edits
     * deep-merge. No-op if this node is not an array holding [id].
     */
    public fun editElement(replica: ReplicaId, id: ElementId, edit: (JsonNode) -> JsonNode): JsonNode {
        val base = arrContent() ?: return this
        val current = base.items.get(id) ?: return this
        return writeSupersede(replica, Content.Arr(base.order, base.items.put(replica, id, edit(current))))
    }
```

```kotlin
/**
 * A CRDT modelling an arbitrary JSON document. The root is always a JSON object.
 * `piece` is the structural lattice join of the two roots; any two replicas that
 * absorb the same set of deltas — in any order, with repeats — converge to the
 * same [value].
 */
public class JsonCrdt internal constructor(internal val root: JsonNode) : Quilted<JsonCrdt> {

    override fun piece(other: JsonCrdt): JsonCrdt = JsonCrdt(root.piece(other.root))

    /** Materialize the whole document as a plain [JsonObject]. */
    public fun value(): JsonObject = root.toJsonElement() as JsonObject

    override fun equals(other: Any?): Boolean = other is JsonCrdt && root == other.root
    override fun hashCode(): Int = root.hashCode()
    override fun toString(): String = "JsonCrdt(${value()})"

    public companion object {
        /** The empty document `{}`. */
        public fun empty(): JsonCrdt = JsonCrdt(JsonNode.emptyObject())

        /** Build a document from a plain [JsonObject], authoring every node as [replica]. */
        public fun fromJson(replica: ReplicaId, json: JsonObject): JsonCrdt =
            JsonCrdt(buildNode(replica, json))

        /** Recursively build a [JsonNode] from any [JsonElement]. `JsonNull` is a scalar. */
        internal fun buildNode(replica: ReplicaId, element: JsonElement): JsonNode = when (element) {
            is JsonObject -> element.entries.fold(JsonNode.emptyObject()) { node, (k, v) ->
                node.put(replica, k, buildNode(replica, v))
            }
            is JsonArray -> element.foldIndexed(JsonNode.arr(replica)) { i, node, v ->
                node.insertAt(replica, i, buildNode(replica, v))
            }
            is JsonPrimitive -> JsonNode.scalar(replica, v)
        }
    }
}
```

```kotlin
private fun segments(path: String): List<String> = path.split("/").filter { it.isNotEmpty() }

/** Set [value] (scalar or subtree) at [path], auto-creating intermediate objects. */
public fun JsonCrdt.set(replica: ReplicaId, path: String, value: JsonElement): JsonCrdt =
    JsonCrdt(setIn(root, segments(path), replica, value))

/** Remove the entry at [path] (object key or array index). No-op if absent. */
public fun JsonCrdt.remove(replica: ReplicaId, path: String): JsonCrdt {
    val segs = segments(path)
    if (segs.isEmpty()) return this
    return JsonCrdt(removeIn(root, segs, replica))
}

/** The materialized value at [path], or null if no node lives there. */
public fun JsonCrdt.get(path: String): JsonElement? = nodeAt(root, segments(path))?.toJsonElement()

/** The conflicting alternatives at [path] (see [JsonNode.conflicts]); empty if unambiguous. */
public fun JsonCrdt.conflicts(path: String): List<JsonElement> =
    nodeAt(root, segments(path))?.conflicts() ?: emptyList()

private fun nodeAt(node: JsonNode, segs: List<String>): JsonNode? {
    if (segs.isEmpty()) return node
    val head = segs.first()
    val idx = head.toIntOrNull()
    val child = if (idx != null && node.arrContent() != null) {
        node.arrVisibleIds().getOrNull(idx)?.let { eid -> node.arrContent()!!.items.get(eid) }
    } else {
        node.child(head)
    } ?: return null
    return nodeAt(child, segs.drop(1))
}

private fun setIn(node: JsonNode, segs: List<String>, replica: ReplicaId, value: JsonElement): JsonNode {
    if (segs.isEmpty()) return JsonCrdt.buildNode(replica, value)
    val head = segs.first()
    val tail = segs.drop(1)
    val idx = head.toIntOrNull()
    if (idx != null && (node.arrContent() != null || node.toJsonElement() == JsonObject(emptyMap()))) {
        // array path: edit existing index, or append at end
        val arrNode = if (node.arrContent() != null) node else JsonNode.arr(replica)
        val ids = arrNode.arrVisibleIds()
        return when {
            idx in ids.indices -> {
                val eid = ids[idx]
                val current = arrNode.arrContent()!!.items.get(eid) ?: JsonNode.emptyObject()
                arrNode.editElement(replica, eid) { setIn(current, tail, replica, value) }
            }
            idx == ids.size && tail.isEmpty() ->
                arrNode.insertAt(replica, idx, JsonCrdt.buildNode(replica, value))
            else -> node // out of range — no-op
        }
    }
    val child = node.child(head) ?: JsonNode.emptyObject()
    return node.put(replica, head, setIn(child, tail, replica, value))
}

private fun removeIn(node: JsonNode, segs: List<String>, replica: ReplicaId): JsonNode {
    val head = segs.first()
    val tail = segs.drop(1)
    if (tail.isEmpty()) {
        val idx = head.toIntOrNull()
        return if (idx != null && node.arrContent() != null) node.removeAt(replica, idx)
        else node.removeKey(replica, head)
    }
    val child = node.child(head) ?: return node
    return node.put(replica, head, removeIn(child, tail, replica))
}
```

Note: `editElement`, `arrContent`, `arrVisibleIds`, `child`, `removeKey`, `removeAt` are all public on `JsonNode` from earlier tasks; `buildNode` is `internal` on `JsonCrdt.companion`. The `node.toJsonElement() == JsonObject(emptyMap())` guard lets a numeric segment create a fresh array on an empty-object node.

```kotlin
package us.tractat.kuilt.crdt

import kotlinx.serialization.KSerializer
import kotlinx.serialization.builtins.MapSerializer
import kotlinx.serialization.builtins.serializer
import kotlinx.serialization.descriptors.SerialDescriptor
import kotlinx.serialization.descriptors.buildClassSerialDescriptor
import kotlinx.serialization.descriptors.element
import kotlinx.serialization.encoding.CompositeDecoder
import kotlinx.serialization.encoding.Decoder
import kotlinx.serialization.encoding.Encoder
import kotlinx.serialization.encoding.decodeStructure
import kotlinx.serialization.encoding.encodeStructure
import kotlinx.serialization.json.JsonPrimitive

/**
 * Hand-written serializers for the [JsonCrdt] tree. They break the mutually-recursive
 * [JsonNode] ↔ [Content] cycle (a generated serializer cannot self-reference) and thread
 * [Rga.wireSerializer] for the `Rga<Unit>` order field to avoid the CBOR
 * `PolymorphicSerializer(Any::class)` trap (see [RgaSerializer]).
 *
 * Wire shape: a [JsonNode] is `{ "dots": [Dot], "vals": Map<Dot, Content> }` (the register's
 * store + context, encoded via the existing `Causal`/`DotFun`/`DotContext` serializers), and
 * [Content] is a tagged union `{ "kind": Int, "payload": <kind-specific> }`.
 */
internal class JsonNodeSerializer : KSerializer<JsonNode> {

    // The recursion fixpoint: Content references JsonNode references Content.
    private val contentSerializer: ContentSerializer by lazy { ContentSerializer(this) }
    private val dotFunSerializer: KSerializer<DotFun<Content>> by lazy { DotFun.serializer(contentSerializer) }
    private val causalSerializer: KSerializer<Causal<DotFun<Content>>> by lazy {
        Causal.serializer(dotFunSerializer)
    }

    override val descriptor: SerialDescriptor =
        buildClassSerialDescriptor("JsonNode") { element("reg", Causal.serializer(DotFun.serializer(ContentMarker)).descriptor) }

    override fun serialize(encoder: Encoder, value: JsonNode) {
        encoder.encodeSerializableValue(causalSerializer, value.reg)
    }

    override fun deserialize(decoder: Decoder): JsonNode =
        JsonNode(decoder.decodeSerializableValue(causalSerializer))

    private companion object {
        // A placeholder descriptor source for the lazy cycle; never used for encode/decode.
        val ContentMarker: KSerializer<Content> = ContentSerializer(JsonNodeSerializerHolder)
        val JsonNodeSerializerHolder: KSerializer<JsonNode> = object : KSerializer<JsonNode> {
            override val descriptor = buildClassSerialDescriptor("JsonNodeRef")
            override fun serialize(encoder: Encoder, value: JsonNode) = error("ref only")
            override fun deserialize(decoder: Decoder): JsonNode = error("ref only")
        }
    }
}

/** Tagged-union serializer for [Content], parameterized by the node serializer (recursion break). */
internal class ContentSerializer(private val node: KSerializer<JsonNode>) : KSerializer<Content> {

    private val objMapSer: KSerializer<ORMap<String, JsonNode>> by lazy {
        ORMap.serializer(String.serializer(), node)
    }
    private val orderSer: KSerializer<Rga<Unit>> = Rga.wireSerializer(Unit.serializer())
    private val itemsSer: KSerializer<ORMap<ElementId, JsonNode>> by lazy {
        ORMap.serializer(ElementId.serializer(), node)
    }

    override val descriptor: SerialDescriptor = buildClassSerialDescriptor("Content") {
        element<Int>("kind")
        element("obj", objMapSer.descriptor, isOptional = true)
        element("order", orderSer.descriptor, isOptional = true)
        element("items", itemsSer.descriptor, isOptional = true)
        element("scalar", JsonPrimitive.serializer().descriptor, isOptional = true)
    }

    override fun serialize(encoder: Encoder, value: Content): Unit = encoder.encodeStructure(descriptor) {
        when (value) {
            is Content.Obj -> {
                encodeIntElement(descriptor, 0, 0)
                encodeSerializableElement(descriptor, 1, objMapSer, value.map)
            }
            is Content.Arr -> {
                encodeIntElement(descriptor, 0, 1)
                encodeSerializableElement(descriptor, 2, orderSer, value.order)
                encodeSerializableElement(descriptor, 3, itemsSer, value.items)
            }
            is Content.Scalar -> {
                encodeIntElement(descriptor, 0, 2)
                encodeSerializableElement(descriptor, 4, JsonPrimitive.serializer(), value.value)
            }
        }
    }

    override fun deserialize(decoder: Decoder): Content = decoder.decodeStructure(descriptor) {
        var kind = -1
        var obj: ORMap<String, JsonNode>? = null
        var order: Rga<Unit>? = null
        var items: ORMap<ElementId, JsonNode>? = null
        var scalar: JsonPrimitive? = null
        loop@ while (true) {
            when (val i = decodeElementIndex(descriptor)) {
                CompositeDecoder.DECODE_DONE -> break@loop
                0 -> kind = decodeIntElement(descriptor, 0)
                1 -> obj = decodeSerializableElement(descriptor, 1, objMapSer)
                2 -> order = decodeSerializableElement(descriptor, 2, orderSer)
                3 -> items = decodeSerializableElement(descriptor, 3, itemsSer)
                4 -> scalar = decodeSerializableElement(descriptor, 4, JsonPrimitive.serializer())
                else -> error("Unexpected index $i in Content deserializer")
            }
        }
        when (kind) {
            0 -> Content.Obj(obj ?: ORMap.empty())
            1 -> Content.Arr(order ?: Rga.empty(), items ?: ORMap.empty())
            2 -> Content.Scalar(scalar ?: error("missing scalar"))
            else -> error("Unknown Content kind $kind")
        }
    }
}
```

3. Path facade has no `~0`/`~1` escaping and supports array append only at `index == size`.
4. Array element removal uses Rga-tombstone (order) + ORMap observed-remove (content); materialization gates on both, so the semantics are "hidden when either drops it" — fine for v1, worth a conformance test later.
